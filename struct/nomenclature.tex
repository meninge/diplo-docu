\nomenclature{FPGA}{Field-Programmable Gate Array : circuits intégrés re-programmables}
\nomenclature{IP}{Intellectual Property: Bloc hardware réutilisable ayant une fonctionnalité spécifique}
\nomenclature{MNIST}{Mixed National Institute of Standards and Technology : Base de données de caractère manuscrit servant de tests à de nombreux algorithmes de reconnaissance d'écriture}
\nomenclature{DSP}{Digital Signal Processing : Composant électronique disponible dans le FPGA permettant de faire des multiplication et addition de façon optimisée.}
\nomenclature{FIFO}{First In First Out : structure de données permettant de stocker des données et de les restitué dans l'ordre d'arrivée.}
